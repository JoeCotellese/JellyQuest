# VirtualMonitorPOC

Experiments on virtual monitor size and positioning for Meta Quest.

Uses the Meta Spatial SDK to render floating panels at real-world screen dimensions (32"–100") and configurable viewing distances (desk to cinema). Right thumbstick controls cycle through size and distance presets to compare how different configurations feel in VR.

## Controls

- **Right thumbstick left/right** — cycle screen size (32" Monitor, 55" TV, 65" TV, 75" TV, 100" Projector)
- **Right thumbstick up/down** — cycle viewing distance (Desk 0.7m, Living Room 2.5m, Theater 5.0m, Cinema 8.0m)

## Requirements

- Meta Quest (2/3/3S/Pro)
- Android Studio with Android SDK 34
- Meta Spatial Editor
